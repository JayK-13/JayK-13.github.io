What is this website?

This website is an archive of book reviews by myself. I'm no professional, but I do enjoy reading a lot and like to (very occcasionally) write a bit.

I probably won't update this often, but I'm starting off with one review and may add more for LA class or because I'm bored.

Why a website?

I created this website for coding class in school. Everyone was required to create a html website, so I decided to make one about book reviews.

I chose to make one about book reviews since I often like to look through book reviews when deciding what to read next and often there aren't that many good ones, so an archive of a lot of them was an appealing idea for me. Enjoy.
